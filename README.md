# Список любимых песен на Express.js

Контрольная работа №5 по дисциплине «Технологии индустриального программирования».  
Вариант №20  **«Список любимых песен»**.

## Функционал

Приложение представляет собой простое API на Express.js и статическую страницу:

- Хранение списка песен (в памяти сервера);
- Получение всех песен с фильтрацией по жанру и исполнителю;
- Получение одной песни по `id`;
- Добавление новой песни;
- Обновление существующей песни;
- Удаление песни;
- Статический фронтенд (HTML + JS), который работает с API;
- Собственный middleware 
- Разнесение логики по модулям
- Раздача статики через `express.static()`.

## Стек

- Node.js
- Express.js
